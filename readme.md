# IPL 2026 Predictor 🏏

### How to update:
1. **Matches:** When a match ends, open your Google Sheet and type the team name in the **Winner** column (Column G).
2. **Phase 2:** Once the BCCI announces the remaining 54 matches, simply add them to the bottom of **Sheet1**.
3. **Leaderboard:** It updates automatically whenever someone opens the page!

**Note:** The site uses `localStorage` for the "Login" system to keep things simple and secure.
